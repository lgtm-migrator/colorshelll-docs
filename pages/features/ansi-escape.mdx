# Ansi Escape Codes

Colorshelll displays colors using [Ansi Escape Codes](https://en.wikipedia.org/wiki/ANSI_escape_code), although some terminals don't support it.
Here is a list of terminals that support Ansi Escape Codes.

1. [Hyper](https://hyper.is/)
2. [Alacritty](https://alacritty.org)
3. [Windows Terminal](https://github.com/Microsoft/Terminal)
4. [Tabby](https://tabby.sh)
